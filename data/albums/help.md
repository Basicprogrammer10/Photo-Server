# Album Help

The Albums can be formatted like this

```
album_<NAME / ID>
 ├── config.cfg
 ├── readme.md
 ├── cover.png
 └── images
      ├── image_one.png
      ├── another_image.png
      └── ...
```

## - config.cfg
```ini
; Base Config
name = <NAME>
author = <PHOTOGRAPHER>
host_path = <PATH> # ex /camping

; File Paths and stuff
cover_path = cover.jpg
readme = readme.md
image_dir = images
```
